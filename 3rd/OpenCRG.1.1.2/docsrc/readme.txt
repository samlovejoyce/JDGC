Welcome to the OpenCRG documentation source. This set of
script files let you visualize the test and dokumention files as well as
formating your own scripts like openCRG dokumentations.

make (Linux):

Makefile to convert latex source code to pdf. A copy is provided in
'<path-to-OpenCRG>/doc/latex'.

The publishing method in Matlab is provided since R2007a.

See '<path-to-OpenCRG>/doc/latex/Makefile' for detail information.


xsl:

Set 'cascading style' under publish configurations to '<path-to-OpenCRG>/docsrc/xsl/openCRG_mxdom2latex.xsl'

Converts xml to latex. This Version differs from the matlab source.
Use this file to get latex documents like provided pdfs in
'<path-to-OpenCRG>/doc/pdf'.
